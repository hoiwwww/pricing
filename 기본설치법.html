1VS Code란 무엇인가요?
📝 한 줄 정의
Microsoft에서 만든 무료 코드 편집기입니다. 문서 작업의 "한컴오피스"처럼, 코드·설정·데이터 파일 작업의 표준 도구로 널리 쓰입니다.
🤔 왜 필요한가요
이번에 배포된 standalone.html 같은 산출물을 열람·수정하고, Claude Code와 함께 대화하며 데이터/화면을 조정하는 작업 창구가 됩니다.
✅ 특징
무료 · 가볍고 빠름 · 한글 지원 · 확장 프로그램(Extension)으로 기능 추가 · Windows/Mac 모두 지원
💡 비유하면: VS Code는 "메모장"의 강화판입니다. 파일을 열고 보고 고칠 수 있고, 필요한 기능(한글 언어팩, Claude Code 등)은 확장 프로그램으로 나중에 추가합니다.
2VS Code 설치하기 (Windows 기준)
1
공식 사이트 접속
웹 브라우저에서 code.visualstudio.com 접속 → 화면 중앙의 "Download for Windows" 버튼 클릭
2
설치 파일 실행
다운로드된 VSCodeUserSetup-x64-*.exe 파일을 더블클릭하여 실행합니다.
3
사용권 계약에 동의
"동의합니다"를 선택하고 다음을 눌러 진행합니다. 설치 위치는 기본값 그대로 두어도 됩니다.
4
추가 작업 선택 화면
"PATH에 추가" 항목은 기본 체크된 상태 그대로 유지합니다 (나중에 터미널에서 명령어로 실행할 때 필요). 바탕화면 아이콘 생성 등 원하는 옵션을 추가로 체크해도 무방합니다.
5
설치 완료 및 실행 확인
설치가 끝나면 자동으로 VS Code가 실행됩니다. 이후에는 시작 메뉴에서 Visual Studio Code를 검색해 실행하면 됩니다.
3한글 언어팩 설치하기
기본 설치 상태는 영문 메뉴입니다. 아래 과정으로 한글 메뉴로 바꿀 수 있습니다 (선택 사항이지만 강력 권장).
1
확장(Extensions) 메뉴 열기
VS Code 왼쪽 세로 아이콘 바(액티비티 바)에서 네모 4개 아이콘을 클릭합니다. 단축키 Ctrl + Shift + X
2
"Korean Language Pack" 검색
검색창에 Korean Language Pack 입력 → Microsoft에서 제공하는 "Korean Language Pack for Visual Studio Code"를 선택합니다.
3
Install 클릭 후 재시작
설치가 끝나면 화면 우측 하단에 Change Language and Restart 알림이 뜹니다. 클릭하면 VS Code가 재시작되며 한글 메뉴로 바뀝니다.
4
(알림이 안 뜰 경우) 수동으로 변경
Ctrl + Shift + P → Configure Display Language 입력 → 목록에서 ko 선택 → 재시작
4Claude Code 설치 및 로그인
1
확장 메뉴에서 검색
Ctrl + Shift + X로 확장 메뉴를 열고 Claude Code를 검색합니다.
2
Anthropic 제공 확장 설치
게시자(Publisher)가 Anthropic으로 표기된 "Claude Code" 확장을 선택하고 Install을 클릭합니다.
3
Claude 아이콘 확인
설치가 끝나면 왼쪽 액티비티 바에 Claude 아이콘이 새로 생깁니다. 클릭해 패널을 엽니다.
4
계정 로그인
"Sign in" 버튼 클릭 → 브라우저가 열리면 사내에서 발급받은 Anthropic 계정으로 로그인 → 권한 승인(Authorize) 후 VS Code로 돌아옵니다.
5
로그인 완료 확인
Claude 패널 하단 입력창에 아무 질문(예: "안녕")을 입력해 응답이 오는지 확인하면 준비 완료입니다.
5배포받은 폴더 불러오기
메일/메신저로 받은 신용v0.2 폴더(standalone.html 포함)를 VS Code로 열어보는 과정입니다.
1
배포받은 폴더 준비
전달받은 압축 파일(.zip)을 원하는 위치(예: 바탕화면)에 압축 해제합니다. 압축이 아니라 폴더째 전달받았다면 바로 다음 단계로 진행합니다.
2
VS Code에서 "폴더 열기"
상단 메뉴 파일(File) → 폴더 열기(Open Folder...) 클릭 → 압축 해제한 폴더를 선택하고 폴더 선택을 누릅니다.
3
탐색기에서 파일 목록 확인
왼쪽 탐색기 패널에 standalone.html을 포함한 폴더 내 파일 목록이 표시되면 정상적으로 불러온 것입니다.
4
실제 대시보드 화면으로 열람
VS Code에서 standalone.html을 클릭하면 코드(내용)가 보입니다. 그래프·카드가 보이는 실제 화면을 확인하려면 해당 파일을 우클릭 → Reveal in File Explorer 선택 후, 열린 파일 탐색기에서 더블클릭해 웹 브라우저로 엽니다.

①
탐색기(Explorer)
현재 폴더의 모든 파일·하위 폴더 목록을 보여줍니다. 가장 자주 쓰는 메뉴입니다.

검색(Search)
폴더 전체에서 특정 단어·문구를 찾을 때 사용합니다.
③
소스 제어(Source Control)
Git으로 변경 이력을 관리할 때 사용합니다 (초기엔 사용 빈도 낮음).
④
실행 및 디버그
코드를 직접 실행/점검할 때 사용합니다 (일반 팀원은 거의 사용하지 않음).
⑤
확장(Extensions)
한글 언어팩, Claude Code 등 추가 기능을 설치하는 곳입니다.
⑥
Claude Code
Claude와 대화하며 파일을 열람/수정 요청하는 채팅 패널입니다.
가운데 편집 영역은 현재 열린 파일 내용을 보여주고, 위쪽 탭으로 여러 파일을 동시에 열어둘 수 있습니다. 하단 터미널 패널은 명령어를 입력하는 공간이며, 맨 아래 상태 표시줄은 현재 파일 인코딩·줄 번호 등 정보를 보여줍니다.
